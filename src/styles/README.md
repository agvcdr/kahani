# Styles

Organize future styles around design tokens and responsive defaults.

Suggested folders:
- `tokens/` - colors, spacing, typography, breakpoints, radii, shadows
- `base/` - reset, base document styles, typography defaults
- `components/` - component-level styles if not colocated
- `utilities/` - layout helpers and reusable utility classes

Prioritize mobile-first CSS and document browser fallback decisions.
