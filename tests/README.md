# Tests

This folder is reserved for local testing once application code is added.

Suggested groups:
- `e2e/` - browser flows such as menu browsing, contact links, directions, and reservations
- `visual/` - responsive snapshots across key viewport widths
- `accessibility/` - automated accessibility checks
- `performance/` - Lighthouse or similar budgets
- `fixtures/menu/` - test menu content for edge cases
