# Accessibility and SEO Checklist

Accessibility:
- Semantic landmarks for header, main, nav, footer, sections, and forms
- Clear heading hierarchy
- Keyboard-visible focus states
- Accessible names for links and buttons
- Form labels and helpful error states
- Alt text for meaningful images
- Decorative images marked appropriately once implemented
- Color contrast checked for all text and key UI states
- No motion-only meaning; honor reduced-motion preferences

Restaurant SEO:
- Unique title and meta description per page
- Local business details are consistent: name, address, phone, hours
- Menu page has crawlable text content
- Open Graph and social preview data
- Structured data planned for restaurant/local business details
- Sitemap and robots configuration planned before deployment
