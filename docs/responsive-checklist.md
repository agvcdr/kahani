# Responsive Design Checklist

Use this checklist for every layout change.

- Mobile-first layout works without horizontal scrolling.
- Primary actions are visible and reachable on small screens.
- Header and navigation do not cover content when sticky.
- Menu categories are easy to scan on mobile.
- Cards or rows adapt gracefully from one column to multiple columns.
- Images preserve aspect ratio and do not cause layout shift.
- Text remains readable at all viewport sizes.
- Long menu item names and descriptions wrap cleanly.
- Touch targets are comfortably spaced.
- Layout works with larger browser font settings.
