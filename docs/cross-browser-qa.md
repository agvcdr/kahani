# Cross-Browser QA Matrix

Target browsers:
- Chromium-based browsers
- Firefox
- WebKit/Safari
- Mobile Safari behavior where possible
- Android Chrome behavior where possible

Viewport groups:
- 320px narrow mobile
- 375px common mobile
- 430px large mobile
- 768px tablet
- 1024px small laptop
- 1280px desktop
- 1440px wide desktop

Check for:
- Navigation usability
- Menu grid/list wrapping
- Text legibility
- Tap target spacing
- Sticky headers or call-to-action behavior
- Image aspect ratios
- CSS feature fallbacks
- Form controls and focus states
- Reduced-motion preferences
- Print or share behavior if needed
